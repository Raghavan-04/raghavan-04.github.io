Low-Power RISC-V Processor

VLSI Design • Spring 2025 • Individual Project

VLSI Verilog RTL Design Low Power

Project Overview

Designed and verified a complete 5-stage pipelined RISC-V processor optimized for power efficiency, targeting embedded applications where battery life is critical. The project involved implementing advanced power-saving techniques including clock gating, dynamic voltage scaling, and intelligent sleep modes.

The Challenge

Modern embedded systems demand increasingly powerful processors while maintaining strict power budgets. Traditional processor designs consume excessive power during idle periods and lack fine-grained control over individual functional units. The challenge was to create a processor that delivers competitive performance while reducing power consumption by at least 40% compared to baseline implementations.

Technical Approach

Architecture Design

Implemented a classic 5-stage pipeline (Fetch, Decode, Execute, Memory, Write-back) with RISC-V RV32I instruction set support. The architecture includes:

  • Hazard detection and forwarding unit for data dependencies
  • Branch prediction with 2-bit saturating counter
  • Separate instruction and data memory interfaces
  • 32 general-purpose registers with zero register hardwired

Power Optimization Techniques

Clock Gating: Implemented fine-grained clock gating at the register and functional unit level. Unused pipeline stages and ALU components are automatically clock-gated based on instruction type, reducing dynamic power by 35%.

Dynamic Voltage and Frequency Scaling (DVFS): Integrated voltage and frequency control logic that adjusts operating points based on workload demands. The processor can switch between three power modes:

  • High Performance: 1.2V @ 500MHz
  • Balanced: 1.0V @ 300MHz
  • Low Power: 0.8V @ 100MHz

Power Gating: Entire functional blocks (multiplier, divider) can be power-gated when not in use, eliminating leakage current in idle components.

Implementation Details

RTL Design

Developed over 8,000 lines of SystemVerilog code with strict coding guidelines for synthesis. Used parameterized modules for scalability and implemented comprehensive assertion-based verification.

Verification Strategy

  • Created extensive testbench with over 500 directed and random test cases
  • Achieved 98% code coverage and 100% functional coverage for RV32I instructions
  • Used industry-standard RISC-V compliance tests
  • Performed gate-level simulation post-synthesis to verify timing

Synthesis and Physical Design

Synthesized using Synopsys Design Compiler targeting 28nm CMOS technology. Floor planning and place-and-route performed in Cadence Innovus. Final design achieved:

  • Core area: 0.85 mm²
  • Maximum frequency: 550 MHz @ 1.2V
  • Power consumption: 42 mW @ 300MHz (typical workload)
